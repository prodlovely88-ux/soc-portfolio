# Network Traffic Analysis

## Русская версия

Анализ сетевой активности, выявление аномалий и расследование сетевых инцидентов.

## Core Skills
- Анализ PCAP-дампов с использованием Wireshark и tshark
- Глубокое понимание TCP/IP, DNS, HTTP/HTTPS, SMB
- Выявление C2-трафика (Command and Control) и beaconing-активности
- Анализ подозрительных сетевых соединений (Sysmon Event ID 3, логи Firewall)
- Идентификация туннелирования и эксфильтрации данных

## English Version

Network activity analysis, anomaly detection, and network incident investigation.

## Core Skills
- PCAP analysis using Wireshark and tshark
- Deep understanding of TCP/IP, DNS, HTTP/HTTPS, and SMB protocols
- Identifying C2 (Command and Control) traffic and beaconing activity
- Analyzing suspicious network connections (Sysmon Event ID 3, Firewall logs)
- Detecting tunneling and data exfiltration
