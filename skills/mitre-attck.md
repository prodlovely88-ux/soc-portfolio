# MITRE ATT&CK

## Русская версия

Использую MITRE ATT&CK для базового сопоставления наблюдаемой активности с тактиками и техниками атакующего.

## Основной фокус

- Execution
- Persistence
- Discovery
- Lateral Movement
- Command and Control
- Initial Access

## Практические навыки

- Сопоставление событий с возможными MITRE-техниками
- Отделение подтверждённых техник от предположительных
- Использование MITRE ATT&CK в incident reports
- Аккуратная формулировка possible mapping, если контекст не подтверждён
