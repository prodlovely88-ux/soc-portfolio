# Threat Intelligence & OSINT

## Русская версия

Сбор контекста по индикаторам компрометации (IoC) и работа с публичными песочницами для безопасного анализа угроз.

## Core Skills
- Валидация и обогащение IP-адресов, доменов и файловых хэшей
- Автоматизация запросов к API Threat Intel платформ (AbuseIPDB, VirusTotal)
- Использование ANY.RUN / Joe Sandbox для безопасного динамического анализа вредоносных файлов и ссылок
- Сбор OSINT-данных по выявленным угрозам для формирования Analyst Report
- Извлечение IoC из сырых логов (Python scripting)

## English Version

Gathering context for Indicators of Compromise (IoCs) and utilizing public sandboxes for safe threat analysis.

## Core Skills
- Validating and enriching IP addresses, domains, and file hashes
- Automating API requests to Threat Intel platforms (AbuseIPDB, VirusTotal)
- Using ANY.RUN / Joe Sandbox for safe dynamic analysis of malware and malicious URLs
- Gathering OSINT data on identified threats for Analyst Reports
- Extracting IoCs from raw logs via Python scripting
