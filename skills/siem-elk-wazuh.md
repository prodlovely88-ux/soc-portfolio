# SIEM & EDR Operations (Wazuh / ELK)

## Русская версия

Практический опыт работы со связкой ELK Stack (Elasticsearch, Logstash, Kibana) и Wazuh в качестве основного инструмента мониторинга и реагирования.

## Core Skills
- Поиск и фильтрация событий в Kibana (KQL - Kibana Query Language)
- Создание кастомных дашбордов для мониторинга специфичных угроз (например, RDP-брутфорс, аномалии аутентификации)
- Триаж алертов от Wazuh-агентов
- Анализ File Integrity Monitoring (FIM) логов
- Написание и тюнинг базовых правил детектирования (Custom Rules) для снижения False Positives

## English Version

Practical experience using the ELK Stack (Elasticsearch, Logstash, Kibana) combined with Wazuh as the primary SIEM/EDR solution for security monitoring.

## Core Skills
- Log searching and filtering using Kibana Query Language (KQL)
- Creating custom dashboards for threat visibility (e.g., RDP brute-force, authentication anomalies)
- Triaging alerts generated by Wazuh agents
- File Integrity Monitoring (FIM) event analysis
- Writing and tuning custom detection rules to reduce False Positives
